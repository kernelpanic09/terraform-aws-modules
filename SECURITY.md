# Security Policy

## Supported Versions

The latest release is supported. Older versions do not receive security fixes.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Please report security issues privately by opening a [security advisory](https://github.com/kernelpanic09/terraform-aws-modules/security/advisories/new) on this repository. Do not file public GitHub issues for security problems.

I aim to acknowledge reports within 7 days.
